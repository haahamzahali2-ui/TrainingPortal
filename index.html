How to add patients & data

1. Add a patient — open 1_Patients in your Google Sheet. Add a new row: PT-0006, enrollment date, year of birth, sex, diagnosis, etc.

2. Log a visit — open 2_Visits. Add a row with a new Visit_ID (VIS-0006…) linked to the Patient_ID.

3. Record vitals — open 3_Vitals. Add BP, weight, O2 sat, etc. for that Visit_ID.

4. Enter lab results — open 4_Labs_Tests. One row per test (A1C, lipid panel, eGFR, etc.) linked to the Visit_ID.

5. Log medications — open 5_Medications. Add active meds with dose, frequency, and start date.

6. Track outcomes — open 6_Outcomes. Record baseline and latest values — the dashboard calculates improvement automatically.

HIPAA reminder: Never enter patient names anywhere in the spreadsheet. Use Patient_ID only. Keep your ID ↔ name mapping in your EHR or a separate access-controlled system.
