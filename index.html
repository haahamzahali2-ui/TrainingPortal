To write data back to Google Sheets, you need to add an OAuth2 Client ID. Reading works without it.
OAuth2 setup — one time, ~3 minutes

1. Go to console.cloud.google.com → your "Clinic Tracker" project.

2. APIs & Services → Credentials → + Create Credentials → OAuth client ID.

3. Application type: Web application. Name it anything.

4. Under "Authorized JavaScript origins" add your GitHub Pages URL: https://yourname.github.io

5. Click Create → copy the Client ID (ends in .apps.googleusercontent.com) → paste above.

6. Go to OAuth consent screen → add yourself as a Test User so sign-in works before the app is verified.
