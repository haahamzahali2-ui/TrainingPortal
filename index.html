Connection
To write data back to Google Sheets, add your OAuth2 Client ID. Reading works without it.
OAuth2 setup — one time, ~3 min

1. Go to console.cloud.google.com → your "Clinic Tracker" project.

2. APIs & Services → Credentials → + Create Credentials → OAuth client ID.

3. Application type: Web application.

4. Under "Authorized JavaScript origins" add: https://haahamzahali2-ui.github.io

5. Click Create → copy the Client ID → paste above → Save & reconnect.

6. OAuth consent screen → add yourself as a Test User.
